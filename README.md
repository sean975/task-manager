# Task Manager

Ứng dụng quản lý công việc đơn giản

## Installation

1. Cài đặt các package
```bash
npm install 
```
2. Chạy chương trình ở development mode
```bash
quasar dev
```
3. Build chương trình (export bên trong thư mục /dist)
```bash
quasar build -m electron
```
